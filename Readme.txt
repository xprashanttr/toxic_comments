Below packages must be installed in order to run this project(There is an "imports" section in notebook):
    matplotlib
    pandas
    numpy
    scipy
    sklearn
    time

Project is implemented on Google Colab (https://colab.research.google.com) platform.

Notebook also has Kaggle setup and authentication steps mentioned ("Kaggle API setup and data download" section in notebook), so that manual data upload of train.csv and test.csv are not required.
Data required for project code execution is downloaded in current directory if steps mentioned in notebook are followed, Just that the person, who is executing the code, should have kaggle API file “kaggle.json” placed in the home folder of its Google drive.
